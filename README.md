HotTubFeedMachine
=================

A feed recorder and playback app for mac OS X

- install CocoaPods
- run #pod install

- using the app
  - put in your feed URL and click "use" button to verify this feed
  - hit the record button to record the feed
  - wait for more things to be implemented
